# Trace-CSE Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a diagnostic trace mode that logs IRO CommonSubs replacement IDs for `--force-no-cse`.

**Architecture:** Reuse the existing mwcc-debug CommonSubs replay hook. Keep trace and veto decisions independent, expose trace mode through the local and remote CLI wrappers, and mark trace runs as diagnostic so cache sync is skipped.

**Tech Stack:** C DLL hook in `tools/mwcc_debug/mwcc_debug.c`, Typer CLI in `tools/melee-agent/src/cli/debug/__init__.py`, pytest regression tests.
